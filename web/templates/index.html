Output format

PIG TXT stores one note per line.

This version writes the eight-column PIG fingering format:

note_id onset_time offset_time spelled_pitch onset_velocity offset_velocity channel finger_number

Right-hand finger numbers are positive and left-hand finger numbers are negative. The piano-roll verifier preserves these signs so the hand remains visible alongside the finger color.

The velocity columns are required by the format. MuseScore MusicXML usually does not include real per-note performance velocities, so this tool uses neutral defaults unless better data is available later.

Reference: PIG Dataset website
